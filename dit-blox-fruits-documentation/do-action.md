# Do Action

### Get Godhuman/Rengoku/True Triple Katana/Hallow Scythe/Cursed Dual Katana/Soul Guitar

Same with each other, will go get the Item.

```lua
["Get <name>"] = <bool>
```

### Awake Current Blox Fruit

Will awake your eaten Blox Fruit when done get all the Item you set [above](do-action.md#get-godhuman-rengoku-true-triple-katana-hallow-scythe-cursed-dual-katana-soul-guitar).

```lua
["Awake Current Blox Fruit"] = <bool>
```

### Get Mirror Fractal

Will go get Mirror Fractal whenever got God's Chalice but not guarantee will get it, you may get killed during the progress Farming for 500 Mobs to spawn Dough King (have Panic Mode under 50% HP enabled in but still).

```lua
["Get Mirror Fractal"] = <bool>
```
