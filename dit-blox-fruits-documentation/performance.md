# Performance

### Hide Map

Remove map/make Map invisible for huge Performance.

```lua
["Hide Map"] = <bool>
```

### Black Screen

Disable Roblox 3D rendering for better Performance.&#x20;

```lua
["Black Screen"] = <bool>
```

### Lock FPS

(For large quantity) Cap your FPS for less CPU usage.\
But Farm speed will be slightly slower.

```lua
["Lock FPS"] = <number>
```

### Ultra Optimize

Disable nearly all the client-side/local scripts, for best script performance possible.\
This feature still in experimental so bugs/errors may occur.

```lua
["Ultra Optimize"] = <bool>
```
