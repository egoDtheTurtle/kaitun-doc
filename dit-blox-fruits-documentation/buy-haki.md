# Buy Haki

### Enhancement/Skyjump/Flash Step/Observation

Same with each other, will go get the Haki.\
Start buy the Haki after obtained Superhuman.\
For the Observation will took longer to get because of Saber Quest.

```lua
["<name>"] = <bool>
```

### Legendary Enhancement

Will buy 3 Legendary enhancement color if have more than 7.5k fragments.\
Compatible with Auto Tushita (Hop Server), will also Hop Server looking for Legendary Haki.

```lua
["Legendary Enhancement"] = <bool>
```
