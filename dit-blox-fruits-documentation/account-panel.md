# Account Panel

## Website

URL: [https://panel.ditbloxfruit.cc](https://panel.ditbloxfruit.cc)\
Login with your Discord (the account got whitelisted).

### Enable

Decided to enable this or not.\
**If you don't use Account Panel please do not enable this.**\
(This take alot of server's memory so if you not using it, please do not enable)&#x20;

```lua
["Enable"] = <bool>
```

### Delay

The delay of POST to server (update account stats), minimum is `300` seconds

```lua
["Delay"] = <number>
```

### Note

If you have large quantity of device, you should put this `Device 1`/`Device 2`/etc for easier to know which Device is Farming that account.

```lua
["Note"] = <string>
```
