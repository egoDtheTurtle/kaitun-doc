# Blox Fruit

### Blox Fruit Sniper

Automatic buy the Blox Fruit you set when it on stock (prioritize highest value/stop when you ate a Blox Fruit).

```lua
["Blox Fruit Sniper"] = <arg>
```

### Main Blox Fruit

Prioritize buy for the Blox Fruit you set, ignore what current Blox Fruit you ate.

```lua
["Main Blox Fruit"] = <arg>
```

### Eat Sniper Blox Fruits

For [Blox Fruit Sniper](blox-fruit.md#blox-fruit-sniper)/[Main Blox Fruit](blox-fruit.md#main-blox-fruit), instead of only buy it from the stock, it will also eat the Blox Fruit if you got them in Inventory.

```lua
["Eat Sniper Blox Fruits"] = <bool>
```

### Example

Example to put Blox Fruits name in the config (work same as [Main Blox Fruit](blox-fruit.md#main-blox-fruit)):<br>

```lua
["Blox Fruit Sniper"] = {
    "Leopard-Leopard",
    "Dough-Dough",
}
```
